The next steps ğŸ”—

  • file –>New Project–>New Directory–>selecting Website using blogdownamong the options. next, type “Your blog directory name” at Directory nameand “Your Hugo theme you liked (==Usual format: Author/subject name==)” at Hugo theme,at last, choose Open in new session–>Create Project

  • now, we already compeleted environment construction and file project for the blog creation

The second way ğŸ”—
